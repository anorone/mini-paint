# Paint

## How to run the app
Clone the repo.  
Run commands:
```shell
$ cd <project folder>
$ npm ci
$ touch .env
```
Then paste the given Firebase config to the newly create .env file.  
Now to start the application run:
```shell
$ npm run start
```
